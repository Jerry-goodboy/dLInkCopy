# dLinkCopy
                                     
这段时间刚好有空,所以尝试了一下chrome插件的开发.
网上有挺多批量复制页面磁链的插件,功能也挺齐全的,还包括各种格式的批量复制.
我做这个最初的想法是可以在列表页就可以把要下载的链接添加到一个待复制的列表,之后再
一起复制到剪切板,然后复制到网盘中.
                                     
## 功能
* 探测页面中的磁力链接,增加添加按钮
* 管理待复制链接
* 复制页面全部磁力链接
